MorseLink アカウント削除のご依頼方法

このページでは、モールス信号学習アプリ「MorseLink」のアカウントと関連データの削除をご希望のユーザー向けに、手続き方法を案内します。

削除対象となるデータ

アカウント削除のリクエストが承認されると、以下のデータが削除されます。

  • Firebase Authentication に保存されているアカウント情報（メールアドレス、表示名など）
  • 学習履歴（ドリル結果、クエスト進捗、エナジー状態など）
  • チャット履歴と AI コーチへの投稿内容
  • 保存済みの通知トークンおよび購読情報

法令で定められた期間の保管が必要なデータについては、必要最小限の範囲でのみ保持します。

リクエスト手順

  1. MorseLink アプリにログインし、「設定 > アカウント」からユーザー ID を確認します。
  2. 下記の専用メールアドレス宛に、件名を「MorseLink アカウント削除希望」とし、本文に以下の内容をご記載ください。
    • ユーザー ID と登録メールアドレス
    • ご本人確認のための簡単な自己照合情報（例: 最近完了した学習モード）
    • 削除を希望する理由（任意）
  3. 当社サポートより 5 営業日以内に受付確認のメールをお送りします。内容に不備がある場合は追加情報をお願いすることがあります。
  4. ご本人確認が完了次第、7 営業日以内にアカウントと関連データを削除し、完了通知をメールでお知らせします。

お問い合わせ先: support@morse-link.com

注意事項

  • 一度削除したアカウントとデータは復元できません。
  • 定期購入をご利用中の場合は、各ストア（App Store / Google Play）で解約手続きを行ってください。
  • 削除完了まではアプリにログインしないようお願いいたします。
